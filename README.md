T.I.T.O - Simple ChatbotA simple chatbot that can greet, define words, search, and perform basic math operations.

Features- Greet and farewell responses
- Define words using Dictionary API
- Search using DuckDuckGo Instant Answer API
- Basic math operations (addition, subtraction, multiplication, division, exponentiation)
- Typing simulation

Usage1. Run python chatbot.py in your terminal
2. Interact with T.I.T.O by typing commands:
    - hi or hello to greet
    - define <word> to define a word
    - google <query> to search
    - 2+2 to perform math operations
    - quit to exit

APIs Used- Dictionary API (https://api.dictionaryapi.dev/)
- DuckDuckGo Instant Answer API (https://api.duckduckgo.com/)

Requirements- Python 3.x
- requests library (install with pip install requests)
